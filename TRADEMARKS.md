# Typely Brand Policy

The Typely name, logo, icons, screenshots, and distinctive visual identity are project branding controlled by meshahid973. The software license does not grant permission to use that branding for a modified or competing product.

You may:

- Refer to Typely by name when discussing, reviewing, or contributing to the project.
- State truthfully that a project is based on or compatible with Typely.
- Use the name when linking back to the official repository.

You may not:

- Release a modified build under the Typely name.
- Reuse Typely logos, icons, or branding for a fork or competing application.
- Suggest that an unofficial build, fork, service, or website is endorsed by or affiliated with Typely.
- Rename a substantially copied version and present it as an independently created product.

Modified builds should use clearly different names and branding, and they must still comply with the PolyForm Shield License 1.0.0.
