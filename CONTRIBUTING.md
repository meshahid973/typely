# Contributing to Typely

Contributions are welcome through issues and pull requests.

## Before contributing

- Keep changes focused and explain what they improve.
- Follow the existing project structure, formatting, and testing requirements.
- Do not include code, assets, fonts, music, or other material you do not have permission to contribute.
- Run the project checks before opening a pull request.
